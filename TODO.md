# TODO

## 项目目标

围绕赛题“三：具有多模态能力的客服智能体设计”，完成一个可交付的多模态客服系统，核心能力包括：

- 基于说明书文本和配图构建多模态知识库
- 支持文本问题与图片输入的联合理解
- 基于 RAG 准确检索相关知识片段与插图
- 支持多轮对话、复杂问题拆解与幻觉抑制
- 提供可评测的 `/chat` RESTful API
- 形成源码、技术文档、验证报告等完整初赛材料

## 当前建议的优先级

1. 先完成知识库清洗、切分和索引，这是后续问答质量的基础。
2. 再完成最小可用的 `/chat` 接口，尽快跑通“输入问题 -> 检索 -> 生成回答 -> 返回图片”主链路。
3. 在主链路稳定后，重点优化多问题拆分、多轮对话和幻觉抑制。
4. 最后集中补齐测试、文档和验证报告，形成可提交版本。
